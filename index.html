Introduccion

Una base de datos relacional es un tipo de base de datos la cual sus tablas se encuentran conectadas entre si y requieren de la otra, sus datos se organizan en tablas con filas y columnas. En cada colummna se especifica un tipo de dato y en las filas el valor atribuido.

Comandos Básicos de SQL

Comando Descripción
SELECT Seleccionar datos de una tabla
INSERT INTO Insertar datos en una tabla
UPDATE Actualizar datos en una tabla
DELETE Eliminar datos de una tabla
CREATE Crea objetos en la base de datos
ALTER Modifica la estructura de un objeto existente
DROP Elimina objetos de la base de datos
TRUNCATE Elimina todas las filas de una tabla de forma mas rapida que DELETE
